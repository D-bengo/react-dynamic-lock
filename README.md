# React Dynamic Clock

A simple React application that displays a dynamic digital clock using `date-fns`. The clock shows the current date and time in a readable format and updates every second.

## Features
- Displays current date and time
- Automatically updates every second
- Styled with CSS gradient and highlighted time

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/react-dynamic-clock.git
2. Navigate to the project folder:
    cd react-dynamic-clock
3. Install dependencies:
    npm install
4.Start the development server:
     npm run dev
